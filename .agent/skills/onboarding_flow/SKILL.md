# Diretrizes de Onboarding Experience

Em um SaaS moderno (*Product-Led Growth*), o usuário deve conseguir se cadastrar, criar seu banco de dados isolado e já sair inserindo seus imóveis sem a necessidade de falar com ninguém (Zero-touch sales).

## Regras de Ouro

1. **Fricção Mínima:** A tela principal de Cadastro de Agência/Imobiliária deve pedir o MÍNIMO de dados iniciais (E-mail, Senha e Nome da Imobiliária). Mais dados (Logotipo, Endereço Fiscal) devem ser pedidos paulatinamente.
2. **Momentos "Ah-ha" Prévios:**
   * Após o cadastro, sugira carregar 3 imóveis pré-cadastrados como exemplo de demonstração ("Seed Data") para eles verem a vitrine online funcionar imediatamente.
3. **Magic Links via Supabase:** Sugira autenticação por Magic-Link (OTP E-mail) além do tradicional Senha, pois aumenta a conveniência para corretores em trânsito e tem menor taxa de atrito.
